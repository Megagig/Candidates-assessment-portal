## User Flow Examples

### Scenario 1: New User Registration â†’ Login Attempt
1. User registers at `/admin/register`
2. Sees "Pending Approval" success page
3. Tries to login at `/admin/login`
4. **Sees:** Yellow alert with pending approval message
5. **Action:** Waits for admin approval email

### Scenario 2: Approved User â†’ Successful Login
1. Admin approves user account
2. User receives approval email
3. User goes to `/admin/login`
4. Enters correct credentials
5. **Sees:** No error, loading state
6. **Result:** Redirected to dashboard

### Scenario 3: Wrong Password
1. User goes to `/admin/login`
2. Enters correct email, wrong password
3. Clicks "Sign In"
4. **Sees:** Red alert with "Invalid email or password"
5. **Action:** Corrects password and tries again
